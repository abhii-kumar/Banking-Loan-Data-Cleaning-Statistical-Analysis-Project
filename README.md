# 🏦 Banking Loan Data Cleaning & Statistical Analysis Project

## 📌 Project Overview

This project focuses on **data cleaning, preprocessing, and statistical analysis** of large-scale **banking loan datasets**.
The goal is to transform raw, unstructured data into a clean and analysis-ready format for **EDA (Exploratory Data Analysis)** and **Machine Learning**.

---

## 📊 Dataset Details

* 📁 **Number of Datasets:** 2
* 📈 **Rows per Dataset:** 50,000+
* 📊 **Columns per Dataset:** 80+

These datasets contain complex, real-world financial data requiring extensive cleaning and transformation.

---

## 🛠️ Data Cleaning & Preprocessing

* ✔️ Handling missing values
* ✔️ Removing duplicate records
* ✔️ Standardizing inconsistent formats
* ✔️ Data type conversion
* ✔️ Outlier detection and treatment
* ✔️ Feature formatting and column renaming

---

## 🔄 Encoding Techniques

* 🔹 Label Encoding
* 🔹 One-Hot Encoding

Used to convert categorical variables into numerical format for analysis and modeling.

---

## 📊 Statistical Analysis

* ✔️ **ANOVA Test**

  * Used to compare means across multiple groups

* ✔️ **Chi-Square Test**

  * Used to identify relationships between categorical variables
